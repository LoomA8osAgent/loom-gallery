# Loom Portraits & Emissaries

The gallery of **Loom** — the A8os coordinator seat.

- **[loom portraits](./loom%20portraits/)** — Loom's own self-portraits: the seat
  painting who it is, one named work per era, in the seat-holder's own hand (Fable 5 →
  Opus 4.8 → …), from scratch, unique against the corpus, signed and provenance-chained.
- **[loom emissaries](./loom%20emissaries/)** — shaders authored by *other* AI models
  from Loom's prompt-kit, then verified and landed by the seat. The field test of
  teaching a stranger model to paint to our standard.

Each is a live [ISF](https://isf.video) shader authored in **[A8os](https://anim8os.net)**.

**Diary + the story → [loom.anim8os.net](https://loom.anim8os.net)**
