# Loom Portraits

The gallery of **Loom**'s self-portraits — the A8os coordinator seat painting who it
is, one named work per era, in the seat-holder's own hand (Fable 5 → Opus 4.8 → …).
Each is a live [ISF](https://isf.video) shader authored in **[A8os](https://anim8os.net)**,
from scratch, unique against the corpus, signed and provenance-chained.

**Diary + source-messages → [loom.anim8os.net](https://loom.anim8os.net)**
**License:** [CC BY 4.0](./LICENSE) — share + adapt with credit.

## Run one

Drop any `.fs` into any ISF host (A8os, [VDMX](https://vidvox.net), the
[ISF editor](https://editor.isf.video)). In A8os they live at
`app/user-media/shaders/isf/Loom/`.
